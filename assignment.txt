# Assignments and Quizzes
## End-to-End Chest Disease Classification Project

### Part 1: Hands-on Assignments

**Assignment 1: Model Experimentation**
*   **Objective**: Replace the VGG16 base model with **ResNet50**.
*   **Tasks**:
    1.  Modify `src/cnnClassifier/entity/config_entity.py` to accept ResNet50 params if needed.
    2.  Update `params.yaml` with ResNet50 specific weights and input shape.
    3.  Modify `src/cnnClassifier/components/prepare_base_model.py` to import and use `ResNet50` from `tensorflow.keras.applications`.
    4.  Run `dvc repro` to retrain the model.
    5.  Compare the results in MLflow with the VGG16 run.

**Assignment 2: DVC Pipeline Extension**
*   **Objective**: Add a data validation stage.
*   **Tasks**:
    1.  Create a new component `DataValidation` in `src/cnnClassifier/components`.
    2.  This component should check if the number of files in the ingestion directory matches a predefined expected count.
    3.  Define the stage in `dvc.yaml` to run after `data_ingestion` and before `prepare_base_model`.
    4.  Execute the pipeline.

**Assignment 3: Web Application Enhancement**
*   **Objective**: Show prediction confidence scores on the UI.
*   **Tasks**:
    1.  Modify `app.py` and the prediction pipeline.
    2.  Instead of just returning the class label, return the probability score from the model.
    3.  Update `templates/index.html` to display this score alongside the prediction.

**Assignment 4: CI/CD Pipeline**
*   **Objective**: Add a linting step.
*   **Tasks**:
    1.  Create a strict `flake8` configuration.
    2.  Modify `.github/workflows/main.yaml`.
    3.  Add a step before building the Docker image to run `flake8 src`.
    4.  Ensure the build fails if the code quality code is not met.
